# P2-03b - Feed Reading Interactions

## Amac

Feed kartinda okuma deneyimini daha kaliteli hale getirmek: dokununca genisleyen metin, pause etkisi ve daha temiz baslik davranisi.

## Alt Gorevler

- [x] Baslik veya metne dokununca kart icinde genis okuma modu ac
- [x] Genis okuma modunda auto-advance progress barini durdur
- [x] Kullanici kart uzerinde basili tuttugunda progress bar pause etkisi versin
- [x] Uzun basliklari ellipsis ile sinirla
- [x] Icerik scroll'unu sadece genis modda aktif ederek feed swipe cakismini azalt
- [x] Expanded okuma modunu popup kart hissinden cikarip ayni gorsel ustunde daha dogal tam-ekran hisse yaklastir
- [x] Kategori filtrelemesini emoji/farkli category label formatlarina karsi normalize et

## Notlar

- Bu turda ayri detay sayfasi acilmadi; okuma deneyimi kart icinde buyutuldu.
- Genis mod aktifken ana feed swipe'i gecici olarak kapatilir; boylece metin scroll'u kart gecisiyle kavga etmez.
